# Shoot Your Shot

A React/Vite basketball upgrade game.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL Vite prints in the terminal.

## Deploy to Vercel

1. Upload this folder to a GitHub repo.
2. In Vercel, choose **Add New Project**.
3. Import the GitHub repo.
4. Use the default Vite settings:
   - Build command: `npm run build`
   - Output directory: `dist`
5. Deploy.

## Notes

- The game starts at 0 points.
- Progress does not save if the page refreshes. It is intentionally a single-run arcade challenge.
